# Postman ERP Simulado

Coleção Postman simulando um fluxo completo de ERP com testes automatizados.

## Módulos
- Cadastro de clientes
- Produtos e estoque
- Pedidos de venda
- Financeiro e cobrança

## Tecnologias
- Postman
- JSONPlaceholder API
- JavaScript (pm scripts)

## Resultado
11/11 testes automatizados — 0 erros

## Como usar
1. Importe `erp-simulado.postman_collection.json` no Postman
2. Importe `dev.postman_environment.json`
3. Selecione o environment **dev**
4. Execute via Collection Runner
